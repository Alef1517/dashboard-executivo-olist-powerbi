# Dicionário de Dados

| Tabela | Descrição |
|---------|-----------|
| Clientes | Informações dos clientes |
| Pedidos | Informações dos pedidos |
| Itens do Pedido | Produtos vendidos em cada pedido |
| Produtos | Cadastro dos produtos |
| Calendário | Dimensão criada para análises temporais |

---

## Principais KPIs

- Receita Total
- Total de Pedidos
- Clientes Únicos
- Ticket Médio
- Percentual de Pedidos Entregues
- Tempo Médio de Entrega
